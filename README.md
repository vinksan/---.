## Ключевые проблемные области для ИИ в добывающей промышленности

### Кейс 1. Анализ нормативной документации (ГОСТы, инструкции, ПБ)
**Ситуация:**  
Насос марки НШ-50 подаёт масло с давлением 0,3 МПа при норме 1,0 МПа.  
Температура корпуса +85°C (норма до +60°C).  
Датчик вибрации показывает 12 мм/с (норма до 4 мм/с).

**Запрос:**  
> Предложи пошаговый алгоритм диагностики причины и возможные неисправности, ссылаясь на физику процессов.

### Кейс 2. Диагностика неисправностей оборудования
**Ситуация:**  
При производстве взрывных работ на карьере запрещается: вести работы при грозе, тумане (видимость менее 50 м), неисправной звуковой сигнализации.

**Запрос:**  
> Выяви все требования к средствам индивидуальной защиты (СИЗ) из данного документа и сформулируй краткую памятку для машиниста буровой установки по действиям в аварийной ситуации.

## Выводы

### Сильные стороны DeepSeek (зарубежного)
- Глубокое объяснение физики процессов — ценно для диагностики оборудования
- Развёрнутые пошаговые алгоритмы с высокой детализацией
- Хорошее понимание технического контекста (насосная тематика)
- Естественный, профессиональный русский язык

### Сильные стороны GigaChat (отечественного)
- Знание российских нормативных документов (ПБ 03-571-03, РЭ НШ-50-3)
- Табличный формат ответов — наглядно и удобно
- Учёт локальной специфики и терминологии РФ
- Практическая ориентированность — готовые инструкции для персонала

**Общий вывод:**  
Оба чат-бота успешно справляются с решением задач добывающей промышленности в рамках выбранных кейсов. Качество ответов высокое, галлюцинаций не зафиксировано.
